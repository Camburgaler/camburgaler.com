Message Transfer Protocol

This project was made as the final assignment for my Systems Programming class during my Fall 2020 semester. It's a program that accepts a binary message and simulates sending the individual bytes over a network by sending the bites across 3 unique processes, sending confirmation of receipt messages, and printing the message once all bites have been transmitted.

Lighthouse Code Health Monitor

Lighthouse was a project that I worked on during my internship at USAA in the Summer of 2020. Lighthouse was an internal-facing app intended to be used by software development managers for the purpose of monitoring health statistics of projects that they oversaw. The project was in its early stages when I worked on it, so the only statistics available at that time were mutation score, code coverage, and when a given branch was last updated, though there were plans to expand the project beyond that. Unfortunately I'm unable to display the project or its code outside of USAA, but my focus on the project was setting up the project's automated CI/CD pipelines and reporting to QTest.
